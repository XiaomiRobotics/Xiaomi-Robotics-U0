# Xiaomi-Robotics-U0 Tokenizer Assets

This directory intentionally does not include tokenizer vocabulary assets.
Place the public Xiaomi-Robotics-U0 tokenizer files here, or point `--tokenizer-dir` at a
HuggingFace tokenizer directory that contains:

- `unis.tiktoken`
- `unis_vision_tokens.txt`
- `tokenization_unis.py`
- `tokenizer_config.json`
